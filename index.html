Foco Bursátil Web

Ahora con código modular y limpio.

Módulo de Configuración

Todas las constantes del sistema de partículas están ahora en un solo lugar: **particles/config.js**.

Lógica Separada

La física y el movimiento están separados de las funciones de dibujo y redimensionamiento (particle.js y renderer.js).

Punto de Entrada

La aplicación inicia con **main.js**, que usa ES Modules para importar la lógica necesaria.
